# Julian Vance: Strategic Portfolio

A specialized, card-based React portfolio designed for high-performance consultants and strategic operations experts. This project demonstrates modern front-end best practices with a focus on business integrity and technical excellence.

## Features

- **Genuine Consultant Branding**: Replaces all generic placeholders with the professional identity of Julian Vance.
- **Card-Based Interface**: Utilizes a highly structured, "genuine" card-based layout for high readability and professional trust.
- **Premium Tech Stack**: React (Functional Components) + Tailwind CSS v4.
- **Subtle Animations**: Framer Motion used for professional scroll reveals and layout transitions.
- **Dark/Light Mode**: Full theme customization with a refined, high-contrast color palette.
- **Performance Focused**: Minimal dependencies and optimized rendering paths.

## Project Structure

```text
/src
  /components
    /layout       # Structural UI (Navbar, Footer)
    /ui           # Atomic elements (Button, Card)
  /hooks          # State management (useTheme)
  /pages          # Composable sections (Hero, About, Services, etc.)
  /utils          # Helper functions (cn.js)
```

## Implementation Highlights

- **Tailwind v4 Integration**: Uses the latest JIT and PostCSS logic for rapid, scalable styling.
- **Modular Sections**: Each page area is encapsulated in a `.section-card` to ensure consistent elevation and focus.
- **Aesthetic Integrity**: Features a grid background and high-quality iconography from Lucide React.
